## ✨ Features

- ⚡ **High performance** - minimal impact on server TPS
- 🔍 **Low false positives** - accurate and reliable checks
- 🧠 **Intelligent detection**
- 🔧 **Highly configurable**
- 📦 **Fully open-source under the MIT License**
